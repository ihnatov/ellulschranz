* Custom Header and page layout options
* Theme Options

== Installation ==


1. Navigate to Appearance > Themes.
2. Click Install Themes and click the Upload button.
3. Navigate to the ellul-schranz.zip file on your computer and click Install Now.
4. Once you activate EllulSchranz, you will see a notification message to install and activate the required and recommended plugins.
5. Go to Install Plugins and install the required plugins.
6. Simply click the Install button on each plugin to install it.
7. Activate the Visual Composer > Role Manager for all the custom post types.
